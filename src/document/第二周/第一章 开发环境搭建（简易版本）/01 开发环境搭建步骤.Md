##开发环境搭建步骤
#### 一. 安装node.js    
1. 下载地址: https://nodejs.org/en/
2. 安装node.js会自带安装npm包，校验是否安装成功此时可以使用如下命令检查
  + 检查npm安装版本号
    > npm -v 
  + 检查node.js安装版本号
    > node -v

#### 二. 由于国内网络原因（PS：这里大家都懂的啦就不多解释了），如果大家碰到下载一些常用的镜像源非常龟慢的情况，大可尝试下配置下加速源。
  + 参考地址:
    > https://developer.aliyun.com/mirror/NPM?from=tnpm
  + 安装淘宝镜像源命令:
    > npm install -g cnpm --registry=https://registry.npm.taobao.org
    + -g：全局安装
    + --registry：指定仓库
  + 使用 cnpm -v 检查安装版本

#### 三. 安装前端依赖包
  + 安装命令：
    > cnpm install -g @vue/cli
  + 安装完毕后可以检查下版本号看是否安装成功
    > vue --version

#### 四. 安装前端快速原型开发包的指令
  + 安装命令：
    > cnpm install -g @vue/cli-service-global
